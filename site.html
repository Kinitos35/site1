О проекте

Цель проекта

Разработка программного обеспечения для управления системой многоматериальной печати на 3D-принтере, включая механику системы смены материала и алгоритмы точного расчета моментов смены материалов в процессе печати.

Актуальность

3D-печать становится не только средством создания прототипов, но и методом конечного производства готовых изделий. Возможность создания изделий с различными свойствами с помощью многоматериальной печати открывает новые горизонты в производстве, но существующее ПО не обеспечивает точного расчета смены материала во время печати.

Ожидаемые результаты

  • Функционирующее ПО для управления многоматериальной печатью
  • Алгоритмы точного расчета моментов смены материала
  • Интеграция с электроникой 3D-принтера
  • Документация по использованию системы

Анализ существующих решений

event28 февраля 2023

Завершен этап анализа существующего ПО для 3D-печати. Выявлены основные проблемы: неточность смены материала, отсутствие гибких настроек, сложность интеграции с различными моделями принтеров. Начата разработка технического задания.

Разработка первого прототипа

event15 апреля 2023

Завершена разработка первого прототипа системы управления сменой материала. Реализованы базовые функции: управление экструдерами, расчет времени смены материала, простейший интерфейс. Начато тестирование на стенде.
